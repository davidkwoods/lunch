microsoft regional directors

The Microsoft Regional Director program has a new logo. In keeping with the spirit of Microsoft's 2012 corporate and product line rebranding, the RD logo was refashioned to reflect their new design asthetic:

  • Monochromatic
  • Segoe typeface
  • Simple layout and icon
  • Semantic HTML

The Logo

There are three sizes of images (all PNGs) to choose from, with each size coming in 2 flavors: white background and transparent background.

  • 336 x 96 pixels
  • 672 x 192 pixels
  • 1344 x 384 pixels

Small

Medium

Large

The really big version of the logo is 1344 x 384 pixels, available in white or transparent backgrounds.

Details

Compare the new logo with the previous version, which was in use for the past decade. It featured the as-of-then corporate typeface with a narrow band of the four primary Microsoft colors.

Then contrast the new logo within the larger Microsoft product family like Office and Xbox.

More details about the logo files

  • Rendered from Adobe Illustrator
  • File size minimized and optimized with OptiPNG
  • Image height is evenly divisble by 16 or 24 pixels for typographical baseline
  • The logo is a 3 Â½ : 1 ratio
  • The screen color is rgb (0, 125, 195) or hsl (202, 100%, 38%)
  • For print, use cmyk (85, 50, 0, 0)
  • Designed by Sean Dolinsky at Vertigo. Some of his early sketches.
